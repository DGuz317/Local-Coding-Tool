# Treat RepoLens as a layered local code intelligence engine

RepoLens v0.5 and later should evolve from a deterministic assistant preflight helper into a layered local code intelligence engine for assistants and developers. The deterministic graph remains the trusted foundation and default MCP remains read-only, while future layers may add parser/resolver improvements, semantic analysis, labeled AI proposals, and explicitly approved active workflows behind separate opt-in surfaces. This keeps the v0.1-v0.4 safety model as the product substrate without making it a permanent ceiling on real-world code intelligence capabilities.
